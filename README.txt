# Random Quote Machine

A simple web app that fetches random quotes from the [Quotable API](https://api.quotable.io) and displays them dynamically on the page. Click the button to get a new quote.

## How to use

1. Open `index.html` in your browser.
2. The page will show a random quote automatically.
3. Click the "Get New Quote" button to fetch another quote.

## Tech Stack

- HTML
- JavaScript (fetch API, DOM manipulation)
